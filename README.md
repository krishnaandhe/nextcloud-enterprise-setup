# Nextcloud Enterprise Setup

## Overview
Enterprise-grade document management and collaboration platform deployed on Windows Server using IIS, PHP, and MariaDB.

## Use Case
Centralized file storage, secure sharing, HR document management, and department-level collaboration.

## Installation
1. Install IIS + PHP
2. Setup MariaDB/MySQL
3. Configure Web Root
4. Set Data Directory
5. Apply Permissions

## Architecture
User -> IIS Web Server -> Nextcloud App -> Database -> File Storage

## Business Impact
Improved document security and centralized access control.
